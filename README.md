# Weather App

Weather REST API using Express.js and HandlebarsJS as templating engine. 

### Instructions
Install dependencies:

    npm install
    
You can install nodemon to run server:

    npm i -g nodemon
    
Run server on nodemon:

    nodemon src/app -e js,hbs
